
### 2. `io_redirections_and_filters/` - I/O Redirections and Text Processing
Scripts demonstrating input/output redirections, filters, and text processing.

**Scripts:**
- **0-hello_world** - Prints "Hello, World" to standard output
- **1-confused_smiley** - Displays confused smiley "(Ã”o)'"
- **2-hellofile** - Displays content of /etc/passwd file
- **3-twofiles** - Displays content of /etc/passwd and /etc/hosts
- **4-lastlines** - Displays last 10 lines of /etc/passwd
- **5-firstlines** - Displays first 10 lines of /etc/passwd
- **6-third_line** - Displays third line of file "iacta"
- **7-file** - Creates file with complex name containing "Best School" text
- **8-cwd_state** - Writes result of ls -la to file ls_cwd_content
- **9-duplicate_last_line** - Duplicates last line of file "iacta"
- **10-no_more_js** - Deletes all .js files in current directory and subdirectories
- **11-directories** - Counts directories/sub-directories in current directory
- **12-newest_files** - Displays 10 newest files sorted newest to oldest
- **13-unique** - Prints only words that appear exactly once (sorted)
- **14-findthatword** - Displays lines containing "root" from /etc/passwd
- **15-countthatword** - Displays number of lines containing "bin" in /etc/passwd
- **16-whatsnext** - Displays lines containing "root" and 3 lines after in /etc/passwd
- **17-hidethisword** - Displays lines in /etc/passwd not containing "bin"
- **18-letteronly** - Displays lines of /etc/ssh/sshd_config starting with a letter
- **19-AZ** - Replaces A->Z and c->e in input
- **20-hiago** - Removes all c and C from input
- **21-reverse** - Reverses its input
- **22-users_and_homes** - Displays users and home directories sorted by users
- **23-empty_casks** - Finds all empty files/directories in current and subdirectories
- **24-gifs** - Lists all .gif files without extensions, sorted case-insensitive
- **25-acrostic** - Decodes acrostics using first letter of each line
- **26-the_biggest_fan** - Displays top 11 hosts/IPs with most requests from TSV logs
