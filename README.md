# ALU Shell Scripting Exercises

This repository contains solutions to ALU shell scripting exercises. Each script demonstrates basic shell commands and concepts.

## Repository Structure
- `basics/` - Contains all shell script exercises
- Each script is numbered and performs a specific task

## How to Use
1. Clone the repository
2. Navigate to the basics directory
3. Make scripts executable: `chmod +x *`
4. Run any script: `./script_name`
`
