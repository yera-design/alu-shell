the files_variables_and_expansions/` - Variables, Expansions & Initialization
Scripts demonstrating shell initialization, variables, expansions, and aliases.

**Scripts:**
- **0-alias** - Creates alias `ls` with value `rm *`
- **1-hello_you** - Prints "hello" followed by current username
- **2-path** - Adds `/action` directory to PATH environment variable
- **3-paths** - Counts number of directories in PATH
- **4-global_variables** - Lists all environment variables
- **5-local_variables** - Lists all local variables, environment variables, and functions
- **6-create_local_variable** - Creates local variable BEST="School"
- **7-create_global_variable** - Creates global variable BEST="School"
- **8-true_knowledge** - Adds 128 to TRUEKNOWLEDGE environment variable
- **9-divide_and_rule** - Divides POWER by DIVIDE (both environment variables)
- **10-love_exponent_breath** - Calculates BREATH^LOVE (both environment variables)
- **11-binary_to_decimal** - Converts binary in BINARY environment variable to decimal
- **12-combinations** - Prints all two-letter combinations (a-z) except "oo"
- **13-print_float** - Prints NUM environment variable with 2 decimal places
- **14-decimal_to_hexadecimal** - Converts DECIMAL environment variable to hexadecimal
- **15-rot13** - Encodes/decodes text using rot13 cipher
- **16-odd** - Prints every other line from input, starting with first line
- **17-water_and_stir** - Adds WATER (base 8) and STIR (base 3) and displays result
